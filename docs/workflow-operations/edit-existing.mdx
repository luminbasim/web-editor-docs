---
title: Editing an Existing Workflow
---

To edit an existing workflow, use the filesystem navigation on the left, below the `Create Workflow` button.

<img style={{ maxHeight: "400px", margin: "auto" }} src="/images/img20.png" />

To navigate to a workflow:

1. Click the name of the company associated with the workflow.

2. Click on the team directory under the selected company. This will list all the teams associated with this company.
   <img
     style={{ maxHeight: "400px", margin: "auto" }}
     src="/images/img21.png"
   />
3. Then click on the team associated with the workflow. This will list all the workflows associated with this team.
   <img
     style={{ maxHeight: "400px", margin: "auto" }}
     src="/images/img22.png"
   />
4. Click on the identifier of your workflow. The metadata of selected workflow should then appear in the center of the page.

5. To begin editing the workflow, click the Edit Workflow button. The editor will navigate you to the graph editor of the workflow.
   <img
     style={{ maxHeight: "400px", margin: "auto" }}
     src="/images/img23.png"
   />
