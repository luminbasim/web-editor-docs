---
title: Creating a New Workflow
---

To create a new workflow, navigate to https://web-editor.onrender.com/home and click the `Create Workflow` button on the top left corner.

<img style={{ maxHeight: "400px", margin: "auto" }} src="/images/img16.png" />

A form will appear consisting of the following fields:

- **Company**: Every workflow is associated with a company (i.e. a customer of Luminai). Use this field to select the company that is associated with this workflow. If the company does not exist, then use the `Create Company` form to create it.

- **Team**: Every company is divided into multiple teams, and currently, each workflow is associated with a team. Use this field to input the name of the team that is associated with this workflow. A team should only consist of alphanumeric characters, underscores, and dashes.

- **Namespace**: Optional field.

- **ID**: The identifier of the workflow. This identifier must be unique within the current (Company, Team). An id should only consist of alphanumeric characters, underscores, and dashes.

- **Display Name**: Optional display name of the workflow. If provided, the display name will be shown to the user in the extension.

- **Description**: Optional description for the workflow.

<img style={{ maxHeight: "400px", margin: "auto" }} src="/images/img17.png" />

After filling out the form, click the `Create` button to create the workflow. Once the workflow is created, the editor will navigate you to the graph editor for this workflow:
